Detailed Description

Pointers to a C++ vector's data, the vector itself, and the size of the vector.

Carries all data necessary to create an instance of NativeArray{T}.

Note
usage of the name "vector" here corresponds to the C++ Standard Library's std::vector object, not the 3D vector described in Vector3D.
