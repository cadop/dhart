Generate all to all paths in C++.

Parameters
graph_ptrA pointer to the graph to generate paths in.
graph_sizeThe number of nodes in the the grpah pointed to by graph_ptr.
cost_typeType of cost to use for path generation. If left blank will use the default cost of the graph
Returns
An array of CVectorAndData with a length equal to the number of nodes in g squared. Paths will be returned in order starting with all paths from node 0, then all paths from node 1, etc. If a path could not be generated between a set of nodes, then CVectorAndData at that index will be invalid.
Precondition
If cost_type is not left as the default, then it must be the name of a valid cost already defined in graph.
Exceptions
KeyNotFoundExceptioncost_type wasn't left as blank, and didn't refer to the name of any cost that already exists in graph.

Generate a path from start to end.

Parameters

startThe ID of the node to start at.
endThe ID of the node to find a path to.
cost_typeThe type of cost to use for generating the path. If left blank, will use the cost that the graph was created with. In the case of the graph generator, the default cost is distance.

Exceptions
