\doxysubsection{Detailed Description}
Deleter for the \mbox{\hyperlink{a01383}{Boost\+Graph}}. 

\begin{DoxyRemark}{Remarks}
\mbox{\hyperlink{a01391}{Boost\+Graph\+Deleter}} is needed by std\+::unique\+\_\+ptr to destroy a \mbox{\hyperlink{a01383}{Boost\+Graph}}; it is not meant to be called directly.

Implementation based on standard library reference for unique pointer \href{https://en.cppreference.com/w/cpp/memory/unique_ptr}{\texttt{ https\+://en.\+cppreference.\+com/w/cpp/memory/unique\+\_\+ptr}}
\end{DoxyRemark}
\begin{DoxyWarning}{Warning}
\mbox{\hyperlink{a01391}{Boost\+Graph\+Deleter}} is only for use with std\+::unique\+\_\+ptr. You do not want to pass the address of a stack-\/allocated \mbox{\hyperlink{a01383}{Boost\+Graph}} to \mbox{\hyperlink{a01391_a9a83be18f095f5e04d287412cc4bd1d5}{Boost\+Graph\+Deleter.\+operator()()}}. Boost\+Graph\+Deleter.\+operator() calls operator delete on the (\mbox{\hyperlink{a01383}{Boost\+Graph}} $\ast$) argument, and passing the address of a stack-\/allocated \mbox{\hyperlink{a01383}{Boost\+Graph}} for said argument will result in undefined behavior.
\end{DoxyWarning}
