\doxysubsection{Detailed Description}
A queue that remembers every object inserted, and prevents the addition of nodes that have already been inserted even if they\textquotesingle{}ve been popped. 

\begin{DoxyInvariant}{Invariant}
All unique nodes can only enter the queue once. After they\textquotesingle{}ve been popped, future attempts to push them will fail.
\end{DoxyInvariant}
\begin{DoxyRemark}{Remarks}
This class was designed specificlaly to manage the \mbox{\hyperlink{a01245}{Graph\+Generator}}\textquotesingle{}s todo list with as few hashmap calls as possible.
\end{DoxyRemark}
