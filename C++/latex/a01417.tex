class nanort\+::\+BVHAccel$<$ T $>$}
Bounding Volume Hierarchy acceleration. 

\mbox{\hyperlink{a01417}{BVHAccel}} is central part of ray tracing(ray traversal). \mbox{\hyperlink{a01417}{BVHAccel}} takes an input geometry(primitive) information and build a data structure for efficient ray tracing({\ttfamily O(log2 N)} in theory, where N is the number of primitive in the scene).
