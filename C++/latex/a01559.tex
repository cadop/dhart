\doxysubsection{Detailed Description}
general exception of the \mbox{\hyperlink{a01659}{basic\+\_\+json}} class 

This class is an extension of {\ttfamily std\+::exception} objects with a member {\itshape id} for exception ids. It is used as the base class for all exceptions thrown by the \mbox{\hyperlink{a01659}{basic\+\_\+json}} class. This class can hence be used as \char`\"{}wildcard\char`\"{} to catch exceptions.

Subclasses\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{a01563}{parse\+\_\+error}} for exceptions indicating a parse error
\item \mbox{\hyperlink{a01567}{invalid\+\_\+iterator}} for exceptions indicating errors with iterators
\item \mbox{\hyperlink{a01571}{type\+\_\+error}} for exceptions indicating executing a member function with a wrong type
\item \mbox{\hyperlink{a01575}{out\+\_\+of\+\_\+range}} for exceptions indicating access out of the defined range
\item \mbox{\hyperlink{a01579}{other\+\_\+error}} for exceptions indicating other library errors
\end{DoxyItemize}

\begin{DoxyNote}{Note}
To have nothrow-\/copy-\/constructible exceptions, we internally use {\ttfamily std\+::runtime\+\_\+error} which can cope with arbitrary-\/length error messages. Intermediate strings are built with static functions and then passed to the actual constructor. 
\end{DoxyNote}
