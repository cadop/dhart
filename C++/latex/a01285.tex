\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename numeric\+\_\+type = float$>$\newline
class HF\+::\+Geometry\+::\+Mesh\+Info$<$ numeric\+\_\+type $>$}
A collection of vertices and indices representing geometry. 

Internally stored as a 3 by X matrix of vertices and a 3 by X matrix for indices. \mbox{\hyperlink{a00405}{Eigen}} is used to manage all matricies and perform quick transformations such as Rotate\+Mesh. More details on \mbox{\hyperlink{a00405}{Eigen}} are available here\+: \href{https://eigen.tuxfamily.org/dox/group__Geometry__Module.html}{\texttt{ https\+://eigen.\+tuxfamily.\+org/dox/group\+\_\+\+\_\+\+Geometry\+\_\+\+\_\+\+Module.\+html}}

\begin{DoxyInvariant}{Invariant}
Will always hold a valid mesh with finite members.
\end{DoxyInvariant}
