Detailed Description

Contains names for the costs of the cost algorithms in the CostAlgrorithms namespace.

Upon running their respective algorithm, the name defined here can be used to index the cost type created for that graph.

Warning
DO NOT TOUCH EXISTING NAMES. Changing these DOES NOT CHANGE THEM IN C++. Make sure any changes are carried over to native code.
